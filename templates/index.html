Caltopo Proxy

Network Link Workaround

https://caltopo.com/m/[map_id]

This service provides support for network-link access to Caltopo mapswithin the Google Earth web interface.google.com/web). Route information is stored by Caltopo and fetched by the Earth client on demand, allowing you to store and edit map routes within Caltopo with a live-updated view in Earth. #### Suggested workflow: 1. Create a [Caltopo Map](https://caltopo.com), add routes, and explicitly save the map. 2. Open the [Earth Web Interface](https://earth.google.com/web) and enable KML import in the settings. 3. Copypasta the Caltopo map url above and download the network link KML. 4. Upload the KML to Earth. 5. Edit routes in Caltopo, scope in Earth.
